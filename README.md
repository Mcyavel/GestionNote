# Miage Note

Système de Gestion des Notes développé avec PHP 8, JavaScript (ES6+) et SQL.

## Structure du Projet
- `/api/` : Endpoints PHP (JSON)
- `/config/` : Configuration et BDD
- `/js/` : Logique Frontend
- `/css/` : Styles
- `/includes/` : Composants HTML

## Installation
1. Configurer la base de données via `config/schema.sql`.
2. Configurer les accès dans `config/db.php`.
3. Lancer un serveur local : `php -S localhost:8000`.
